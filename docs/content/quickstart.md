# 快速开始

五分钟上手 Memmy。

## 1. 下载安装

从 [官网首页](../) 下载适合你系统的安装包（macOS / Windows），双击安装。

## 2. 注册登录

打开 Memmy，注册一个账号。注册即送 3000 万 Token，无需 API Key，开箱即用。

## 3. 授权扫描

Memmy 会询问你是否允许它读取本机 AI 工具的对话历史。你可以选择：

- **全部允许**：扫描 + 允许向其他 Agent 注入记忆
- **仅允许扫描**：只读取，不主动向其他工具注入
- **不允许**：跳过扫描，手动使用

## 4. 查看初见报告

授权后，Memmy 会扫描你的 Cursor、Claude、Codex 等工具的本地对话，生成一份初见报告——告诉你它了解到了什么，并给出一个可以继续的真实任务。

## 5. 开始使用

你可以：

- 在桌面上直接和 Memmy 对话，交代任务
- 在「记忆管理」页面查看和管理所有记忆
- 在「接入源管理」中添加或移除 Agent 来源
