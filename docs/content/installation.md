# 下载与安装

## macOS

1. 从官网下载 `.dmg` 安装包
2. 双击打开，将 Memmy 拖入 Applications 文件夹
3. 首次打开时，系统会提示"来自未知开发者"——右键选择"打开"即可

**系统要求**：macOS 12 (Monterey) 及以上，支持 Apple Silicon 和 Intel。

## Windows

1. 从官网下载 `.exe` 安装包
2. 双击运行安装程序，按提示完成安装
3. 安装完成后从开始菜单或桌面快捷方式启动

**系统要求**：Windows 10 及以上。

## 首次使用

安装完成后打开 Memmy，它会引导你完成：

1. **注册 / 登录**——注册即送 3000 万 Token
2. **授权扫描**——选择允许 Memmy 读取哪些 AI 工具的本地对话历史
3. **初见报告**——扫描完成后，Memmy 会生成一份关于你的报告，并给出第一个可以继续的任务
