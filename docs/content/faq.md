# 常见问题

## Memmy 和 Cursor / Claude 自带的记忆有什么区别？

Cursor、Claude 等工具自带的记忆只在各自平台内有效。你在 Cursor 里说过的偏好，Claude 不知道；你在 Claude 里做过的决策，Codex 也不知道。Memmy 打通了这些孤岛，让记忆可以在所有 AI 之间流动。

## 记忆数据会上传到云端吗？

不会。所有记忆存储在你本地设备的 `~/.memmy/` 目录中。Memmy 不会将你的记忆数据上传到任何远程服务器。

## 支持哪些 AI 工具？

目前支持 Cursor、Claude Code、Codex 等工具的对话历史扫描。更多工具持续接入中，你也可以手动添加自定义 Agent 来源。

## 免费吗？

免费使用，注册即送 3000 万 Token。你也可以填入自己的大模型 API Key，完全免费。

## 支持哪些操作系统？

目前支持 macOS 12+ 和 Windows 10+。

## 可以删除记忆吗？

可以。你可以在「记忆管理」中搜索、查看、删除任何一条记忆，也可以批量删除或关闭某个来源。

## 开源吗？

是的，Memmy 完全开源。你可以在 [GitHub](https://github.com/memmy) 查看源码。
