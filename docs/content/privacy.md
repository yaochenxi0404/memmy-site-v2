# 数据安全与隐私

Memmy 的设计原则是：**你的记忆，只属于你。**

## 本地优先

所有记忆数据存储在你自己的设备上（`~/.memmy/` 目录），不会上传到任何远程服务器。

## 授权机制

- Memmy 不会在你不知情的情况下读取任何 AI 对话历史
- 每一次扫描都需要你明确授权
- 你可以随时在「记忆管理 → 接入源管理」中调整或撤回授权

## 记忆使用控制

- 每条记忆可以被哪个 Agent 使用，完全由你决定
- 传递多少上下文、传给哪个 AI，每次都需要你确认
- 密钥、凭证等敏感信息会被自动脱敏

## 数据管理

你可以随时：

- 搜索、查看所有记忆
- 删除任意一条或批量删除
- 关闭某个来源的采集
- 导出你的全部记忆数据
